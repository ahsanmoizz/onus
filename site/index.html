The Problem

AI coding agents operate with the same permissions as the user. One bad command wipes hours of work — or worse.

How It Works

Intercept → Evaluate → Decide

Onus sits between "agent decides" and "agent does." Every action goes through the policy engine before execution.

1

Agent Acts

Agent runs a shell command, writes a file, makes an API call — or any other action.

2

Onus Intercepts

The action is evaluated against 13+ safety and scope rules. Regex patterns, path analysis, magnitude checks — all in under 250µs.

3

Verdict Returned

One of four outcomes. If blocked, a correction prompt guides the agent toward a safer approach.

ALLOW
Proceed silently
WARN
Proceed with warning
BLOCK
Halt with correction
ESCALATE
Require human approval

Detection Tiers

Three layers of safety

From instant deterministic rules to cross-deployment anomaly detection — the protection compounds over time.

Tier 1

Deterministic

13 safety and scope rules compiled at startup. Sub-250µs evaluation. Ships in v0.

  • Destructive filesystem operations
  • Sudo abuse and curl-to-bash
  • Environment exfiltration
  • Workspace boundary enforcement
  • Git destructive operations
Tier 2

Heuristic

Magnitude analysis and goal drift detection. Flags oversized diffs and scope creep.

  • Large diff / batch delete detection
  • Working directory drift
  • Unusual file type patterns
  • Command complexity scoring
Tier 3

Learned

Cross-deployment anomaly detection. The compounding moat — every deployment makes the network smarter.

  • Pattern-of-life modeling
  • Cross-org threat intelligence
  • Behavioral clustering
  • Real-time risk scoring
Get Started

Install in seconds

Download the binary, choose your integration, you're protected.

curl -fsSL https://github.com/Gitlawb/onus/releases/latest/download/install.sh | bash

Claude Code (1 line)

Add to your Claude Code settings.json:

"preToolUse": { "command": "onus", "args": ["evaluate"] }

MCP Proxy

Replace your MCP server command with the Onus proxy:

{ "command": "onus", "args": ["mcp-proxy", "--server", "/path/to/real-server", "--", "arg1", "arg2"] }

Shell Wrapper (one-liner)

Source the shell wrapper for terminal-based agent protection:

onus shell install && source ~/.bashrc

Python

pip install onus
